# 可以根据链接下载视频

> 运用request和fs库 
 - download.js / idownload.js 多文件下载
 - single.js 单文件下载

## 使用

下载
```bash
git clone
``` 

安装
```bash
npm install
```

运行
```bash 
node download.js
```

```bash 
node idownload.js
```

```bash 
node single.js
```
